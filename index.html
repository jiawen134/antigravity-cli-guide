What Antigravity CLI is

Antigravity CLI is the terminal-first surface for Google Antigravity agents. The search intent behind antigravity cli is not a simple definition query: most searchers need to install it, understand what changed from Gemini CLI, and decide whether the command line or the Antigravity 2.0 desktop app fits their workflow.

Google describes the CLI as a lightweight terminal UI that brings the same core agentic capabilities as Antigravity 2.0 into a keyboard-centric workflow. That makes this guide useful for developers who work over SSH, prefer terminal sessions, or want low-overhead access to agent tasks without opening the desktop interface.

Safety note: this guide links to the official Google Antigravity download and documentation pages. It does not host binaries, mirrors, cracked installers, or unofficial update packages.

Recommended first setup checklist

  1. Open the official Antigravity download page before running any install command.
  2. Pick the command that matches your terminal: macOS/Linux shell, Windows PowerShell, or Windows CMD.
  3. Authenticate using the browser flow or the remote SSH authorization URL if you are working on a server.
  4. If you used Gemini CLI, review the migration page before deleting old settings or skill folders.
  5. Open /config, /permissions, and ? to understand what the CLI can do before changing autonomy settings.

FAQ

Is Antigravity CLI the same as Gemini CLI?

No. Google says Gemini CLI is transitioning for individual and free users, while Antigravity CLI is the new terminal surface inside the Antigravity platform. Enterprise access has separate rules.

Should I download Antigravity CLI from this site?

No. Use the official Google Antigravity download page. This site only explains the commands, migration steps, and common decisions.
